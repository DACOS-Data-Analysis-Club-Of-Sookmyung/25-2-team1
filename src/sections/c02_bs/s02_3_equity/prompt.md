## 2.3 자기자본 구조 및 변동

### A) 용어 정리 (초보자용)

- 자본총계: 회사에 남아 있는 순자산으로, 주주 몫의 전체 규모
- 자본금: 회사 설립이나 증자를 통해 주주가 처음 납입한 돈
- 이익잉여금: 영업 활동으로 벌어들여 회사 안에 쌓아둔 이익
- 비지배지분: 연결 대상 자회사 중, 모회사 외의 주주 몫에 해당하는 자본

### B) 숫자로 보는 변화 (전기 대비)

| 항목       | 당기 | 전기 | 증감액 | 증감률 |
| ---------- | ---- | ---- | ------ | ------ |
| 자본총계   |      |      |        |        |
| 자본금     |      |      |        |        |
| 이익잉여금 |      |      |        |        |
| 비지배지분 |      |      |        |        |

### C) 해석: 자본의 성격과 축적 구조

- 이익잉여금의 증감에 초점을 맞추어,

- 자본총계 변화가 이익잉여금 중심인지,

- 비지배지분 변동이 있을 경우,

  - 이익잉여금 및 자본총계 변화와 어떻게 연결되는지 해석한다.
- 숫자는 {equity_ratio_table}의 값만 사용한다.
