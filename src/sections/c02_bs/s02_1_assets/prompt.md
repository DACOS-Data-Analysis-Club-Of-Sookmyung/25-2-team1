## 2.1 자산 구조 및 변동

### A) 용어 정리 (초보자용)

- 자산총계: 회사가 보유한 전체 자원의 합
- 유동자산: 1년 이내에 현금으로 바꾸거나 사용할 수 있는 자산
- 비유동자산: 1년 이후에 사용하거나 처분되는 자산
- 현금및현금성자산: 바로 사용 가능한 현금과 이에 준하는 자산
- 매출채권: 제품·서비스를 판매하고 아직 받지 못한 외상대금
- 재고자산: 판매를 위해 보유 중인 상품이나 원재료
- 유형자산: 영업 활동에 사용하는 토지, 건물, 설비 등
  ※ 제공되지 않은 항목은 정의하지 않는다.

### B) 숫자로 보는 변화 (전기 대비)

| 항목             | 당기 | 전기 | 증감액 | 증감률 |
| ---------------- | ---- | ---- | ------ | ------ |
| 자산총계         |      |      |        |        |
| 유동자산         |      |      |        |        |
| 비유동자산       |      |      |        |        |
| 현금및현금성자산 |      |      |        |        |
| 매출채권         |      |      |        |        |
| 재고자산         |      |      |        |        |
| 유형자산         |      |      |        |        |

### C) 해석: 자산 항목별 변화 의미

- 아래 항목에 대해서만 분석한다:
  현금및현금성자산, 매출채권, 재고자산, 유형자산
- 각 항목별로 다음 순서로 설명한다:
  1. 관찰: 전기 대비 증가 또는 감소 여부와 변화 폭
  2. 의미: 초보자 관점에서 시사하는 바
  3. 주석 근거가 있을 경우에만 원인 설명 + 근거 표시

### D) 효율성 지표로 본 보완 해석

- 매출채권회전율(ar_turnover), 재고자산회전율(inventory_turnover)에 대해:
  - 당기와 전기 간 변화 방향을 설명한다.
  - 매출채권 또는 재고자산의 증감과 어떻게 연결되는지 해석한다.
- 숫자는 {asset_efficiency_table}의 값만 사용한다.

### E) 이번 섹션에서 참고한 주석 목록

- note_no를 오름차순으로 나열한다.
- 가능할 경우, 주석 제목이나 핵심 주제를 근거 chunk에서 유추해 한 줄로 설명한다.
