## 3.1 매출 및 원가 구조

### A) 먼저 용어 정리 (초보자용)

- 매출액: 회사가 제품이나 서비스를 팔아 벌어들인 전체 금액
- 매출원가: 제품이나 서비스를 제공하기 위해 직접 들어간 비용
- 매출총이익: 매출액에서 매출원가를 뺀 금액으로, 본업에서 1차적으로 얼마나 남겼는지를 보여주는 지표
- (옵션) 사업부문/부문별 매출: 회사가 어떤 사업 영역에서 매출을 올리고 있는지를 보여주는 정보
  ※ 제공된 경우에만 설명한다.

### B) 숫자로 보는 변화 (전기 대비)

(표는 {is_rev_cost_table} 그대로 사용)

### C) 해석: 매출·원가 구조가 의미하는 바

- 매출액, 매출원가, 매출총이익만 분석한다.
- 각 항목별로:

  1. 관찰(증가/감소와 변화 폭)
  2. 의미(초보자 관점)
  3. 근거가 있을 때만 원인 설명 + 근거 표시
     - 없으면 “주석 근거가 부족해 구체 원인 단정은 어렵다”고 명시

- (옵션) 사업부문 정보가 제공된 경우에만:
  - 특정 부문이 매출 변화에 영향을 준 정황을 근거 기반으로 설명한다.
  - 비율 계산은 하지 않는다.

### D) 주석을 어떻게 활용했는지 (투명성)

- [항목] → [참고 주석 note_no] → [주석에서 확인한 핵심 포인트(1~2줄)] → [해석]
- 주석 근거가 부족하면 “이 항목은 주석 근거가 제한적”이라고 명시한다.
